# Acalc

Academic Score Calculation program

## Description

### Fitting

* Input: Standard Score
* Output: Percentage
* Parameter: $m, \sigma$
    * Standard score : $s = 20 \times \dfrac{x-m}{\sigma} + 100$
    * Output: $\displaystyle \int_{x}^\infty p(t)dt$, where $p(t)\sim \mathcal{N}(x|m,\sigma^2)$
